# Changelog

## 0.1.0 - 2026-04-27
- Initial release of `ash_binance_api_futures`.
- Support for Binance USD-M Futures REST API (Account, Trade, Market Data).
- Support for Binance Futures WebSockets (Streams and API).
- Full coverage of positions, leverage, and margin type management.
- Integration with `ash_binance_api_core` for robust transport.
